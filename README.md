## About The Project
Another boilerplate for node, typescript, jest and eslint.

## Getting Started
Download or clone it.

### Prerequisites
You need npm.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
2. Install NPM packages
   ```sh
   npm install
   ```
4. Run
   ```sh
   npm run start
   ```
